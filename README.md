# PetWatch

## About:
PetWatch is an autonomous security system exclusive to run on the PetMate device.
It uses computer vision and pattern recognition with deep learning on the cloud to not only recognise pets, but also interact with them in unique ways. This includes detecting when a pet sits or falls asleep to respond accordingly - giving a treat or moving less to avoid making noise, for instance.

## Resources:
The main programming language of PetWatch is Python, paired with the OpenCV library to process the live video feed coming from the PetMate camera. 
Multiple Google Cloud services are used in this project:
- Google Cloud Vertex AI
- Google Cloud Pub/Sub
- Cloud Run
- Cloud Storage
- Cloud Firebase
